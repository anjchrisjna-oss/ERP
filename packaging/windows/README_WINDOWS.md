# ERP local - Paquete de pruebas Windows

Este paquete permite probar la UI local y API en Windows sin instalar Tauri.

## Contenido
- `run_erp_local.bat`: arranca la app local en `http://127.0.0.1:8765`
- scripts de migración/seed
- backend + UI

## Requisitos en Windows
- Python 3.10+
- `sqlite3.exe` en PATH

## Uso
1. Ejecuta `run_erp_local.bat`
2. Abre `http://127.0.0.1:8765`
